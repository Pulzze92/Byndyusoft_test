Тестовое задание для компании Byndyusoft

Данное приложение создано для того, чтобы выполнить задание от компании Byndyusoft. Ниже на странице можно найти форму с кнопкой, что и является результатом выполнения данного задания.

На странице есть форма с текстовым полем и кнопкой. В текстовое поле вводят цифры через запятую. После нажатия на кнопку или нажатия клавиши Enter отображается результат обработки введенного массива чисел. Напиши функцию, на вход которой приходит массив чисел. Функция возвращает сумму двух минимальных элементов массива. Например, если дан массив [4, 0, 3, 19, 492, -10, 1], то результатом будет -10, потому что два минимальных числа -10 и 0, а их сумма -10. Напиши минимум 3 модульных теста на эту функцию, проверь, что по клику на кнопку сообщение будет выведено. HINT: учти, что массив может быть пустым, или без цифр или состоять из 100 млн. элементов, поэтому надо учесть разные граничные условия. Не спешите – главный ориентир всегда – качество.

Описание решения

Данное задание было выполнено следующим образом:

  • В начале были найдены форма, поле ввода input и кнопка button с помощью метода querySelector.
  • Далее была написана функция minValues, которая в начале проверяет основные возможные ошибки при ожидании вводимых из поля параметров. Были предусмотрены ситуации с пустым полем ввода (вывод в блок ниже формы текста с ошибкой), с единственным введенным параметром в поле ввода, с введенными кроме чисел символов (предусмотрена ошибка), со слишком большим количеством вводимых параметров (было выбрано 15 для эстетического вида самого поля после ввода такого количества параметров).
  • После проверки вышеописанных условий функция с помощью метода forEach пробегается по (заранее разбитому с помощью метода split) массиву элементов пришедшему из строки input. Внутри перебора проверяется, вводились ли квадратные скобки, если да, то выдается ошибка. Также, проверяется, были ли введены иные символы кроме букв. В остальных случаях эелементы пушатся (преобразованные в числа) в заранее подготовленный новый массив, поле для вывода ошибок очищается.
  • Далее внутри той же функции результирующий массив сортируется с помощью метода sort и в результирующее поле попадает сумма первых двух элементов отсортированного массива. Поле ошибок очищается.
  • Следующая функция отслеживает событие нажатия на кнопку enter. При данном событии запускается вышеописанная функция.
  • Также, все та же функция запускается при событии нажатия на кнопку "Подтвердить" из формы.
  • Последняя функция отслеживает событие изменения поля воода, для того, чтобы очищать поле с ошибками и поле ввода (для комфортного запуска следующих итераций проверки чисел в поле ввода).
  • Данное задание выполнено на сверстанной странице (с минимальными эффектами и анимацией). Использовался чистый HTML, SASS и JavaScript.
